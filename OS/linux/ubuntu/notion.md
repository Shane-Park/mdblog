## 7Zip 설치

> 7Zip은 LZMA 압축 알고리즘을 기반으로 개발된 훌륭한 압축 프로그램 입니다. 개인이나 단체 모두 무료로 사용 할 수 있습니다. 자세한 정보는 https://www.7-zip.org/7z.html 를 참고해주세요.

```zsh
sudo add-apt-repository universe
sudo apt update

```

```zsh
sudo apt install p7zip-full p7zip-rar
```
